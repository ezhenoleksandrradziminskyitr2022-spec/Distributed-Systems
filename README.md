# Getting-things-ready
Даний документ описує перше лабораторне (семінарське) завдання з курсу розподілених систем. 
Його основна мета — підготувати середовище розробки та ознайомити студента з базовими інструментами, які використовуються в сучасній IT-індустрії.

Програма повинна коректно обробляти наступні команди:

1. ping

Не має параметрів

Використовується для перевірки роботи програми

2. echo <text>

Приймає текстовий параметр

Виводить переданий текст

3. login <login> <password>


Приймає логін і пароль

Імітує процес входу в систему

4. list

Не має параметрів

Імітує отримання списку об’єктів або користувачів

5. msg <destinationUser> <text>

Приймає ім’я користувача та текст повідомлення

Імітує надсилання повідомлення

6. file <destinationUser> <filename>

Приймає ім’я користувача та назву файлу

Імітує передачу файлу

7. exit

Завершує виконання програми
